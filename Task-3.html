HTML

1)What are Semantic Elements?

A semantic element clearly describes its meaning to both the browser and the developer.

Examples of semantic elements:

→ <form >  <header > ,  <h1 > to <h6 >

  <section>,  < footer>,   < p>

2)What are non-Semantic Elements?

A non-semantic element it is not describe the content

 Examples of non-semantic elements:

→   <Div > ,  <span >

3) Formatting Elements

Formatting elements were designed to display special types of text:
< b > - Bold text,     < strong >-Important text ,  < i > -Italic text
< em > -Emphasized text,   < sub >- Subscript text,   < sup >-Superscript text
